# 1. 理论基础
- 核心概念：坐标系定义（i系-地心惯性、e系-地心地固、n系-导航如东北天、b系-载体）、欧拉角、方向余弦矩阵、四元数。
- 核心原理：地球模型（如WGS-84）、重力模型、哥氏定理、比力方程（一切导航解算的源头）。
- 误差分析基础：INS误差方程（Φ角或Ψ角误差模型），理解舒勒回路、傅科周期、地球周期。

# 2. 前端预处理与补偿
1. IMU误差标定与补偿算法
	- 六位置法/多位置法标定：用于标定加速度计和陀螺仪的零偏、尺度因子和安装非正交误差（即标定矩阵）。
	- 温度补偿算法：建立零偏、尺度因子与温度的查找表或多项式模型。
	- 在线标定/自适应标定：在组合导航滤波器中，将传感器误差参数（如零偏）作为状态量进行实时估计。
2. 初始对准算法
	- 解析式粗对准：在静基座下，利用重力矢量和地球自转角速度矢量在载体系的投影，直接解析计算初始姿态矩阵（俯仰、横滚、航向）。
	- 卡尔曼滤波精对准：在粗对准的基础上，以姿态误差、速度误差、传感器零偏等为状态量，以外部速度/位置观测（通常为零）为量测，进行滤波迭代，获得高精度初始姿态。这是最主流的方法。
	- 传递对准：适用于子惯导（如飞机上的导弹）在动基座（母机）上的对准，利用母惯导信息进行匹配（速度匹配、位置匹配、姿态匹配等）。
3. IMU数据预处理算法
	- 同步与插值：处理多传感器（如多个IMU或IMU与相机）数据时间戳同步问题。
	- 降噪滤波：对于低精度IMU，可能需要对原始数据进行低通滤波，但需谨慎以免引入延迟。
	- 单位与坐标系转换：将原始脉冲计数或电压值转换为物理量（°/s, m/s²），并统一到约定的载体坐标系。

# 3. 惯性导航算法
1. 姿态更新算法
	- 核心：四元数微分方程更新。（离散化常用一阶或高阶龙格库塔法积分）
	- 补偿：旋转矢量算法。当载体做锥运动时，简单积分会引入“不可交换性误差”。旋转矢量法（如二子样、三子样算法）通过对一个周期内的角增量进行优化（圆锥补偿），显著提高高动态下的姿态精度。
	- 等效转换：最终将更新后的四元数转换为DCM和欧拉角。
2. 速度更新算法
	- 核心：比力方程积分。
		- 坐标转换
		- 有害加速度补偿：减去哥氏加速度和向心加速度
		- 重力补偿：加上重力矢量（包括正常重力和离心力）
	- 积分方法：通常采用梯形法或更高阶积分方法，在姿态更新周期内进行。
3. 位置更新算法
	- 核心：速度积分。
	- 经纬高更新（微分方程）
	- 在解算周期内对上述微分方程进行积分，更新经纬高。

# 4. 组合导航算法
1. 基于滤波的融合算法
	- 线性/扩展卡尔曼滤波器：最经典、应用最广的框架。状态量通常包括：位置误差、速度误差、姿态误差（或失准角）、陀螺零偏、加速度计零偏等。松组合和紧组合的核心区别在于观测模型。
		- 松组合：观测量为位置和速度的差值（INS输出 - GNSS输出）。模型简单，易于实现。
		- 紧组合：观测量为伪距、伪距率（多普勒）的差值。能更好地利用GNSS原始信息，在卫星数不足、抗干扰方面有优势。
	- 无迹卡尔曼滤波器/容积卡尔曼滤波器：另一种处理非线性系统的方式，通过一组确定的采样点（Sigma点）来逼近状态分布，无需计算雅可比矩阵，在某些强非线性场景下表现优于EKF。
2. 基于优化的融合算法
	- 因子图优化：近年来在VIO/VINS领域大放异彩。它将所有状态（位姿、速度、零偏等）和所有观测约束（IMU预积分约束、GNSS约束、视觉重投影约束等）构建成一个图模型，通过非线性优化一次性求解最优状态轨迹。特别适合滑动窗口内的多传感器、多约束融合，精度高，但计算量也大。
3. 其他融合架构
	- 互补滤波器：在姿态层级融合IMU（高频）与磁力计/航向参考（低频）的经典轻量方法，广泛用于无人机和机器人。
	- 联邦滤波器：用于多套INS或多种导航系统的高可靠、容错组合架构，多见于航空航天领域。
