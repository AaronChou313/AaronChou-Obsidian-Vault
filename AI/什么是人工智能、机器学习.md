# 1. 人工智能的分类
人工智能是一个很宏大的概念，目的是实现“非人类智能体”，也就是让机器表现出像人类一样的智能行为，如推理、学习、规划、感知、理解语言等。为了研究这些不同的功能，诞生出了如下几种不同的研究领域：
1. 计算机视觉：让机器人能看见
2. 自然语言处理：让机器人能听懂人类的话
3. 知识表示与推理：研究如何用计算机来表示和处理“知识”
4. 自动规划与调度：让机器人在给定的目标和约束下自动规划步骤，并优化资源分配
5. 搜索和优化：寻找某个问题的最优解决方案
6. 机器学习：AI的一个核心子领域，让计算机从数据中自动学习规律和模式，并利用这些模式对未知数据进行预测或决策，而无需进行明确的、硬编码的程序指令。

早期的AI基本都是依赖规则系统，也就是程序员设计好程序运行的规则，从而让机器完成相应的任务。例如规定60分以上为合格，60分以下为不合格，机器就能按照这个规则快速判断完一万张试卷，哪些是合格的，哪些是不合格的。
但这样的方式只适用于单一模式的任务，什么意思呢？如果我现在有一万张照片，其中包含了猫、狗、兔子、男孩、女孩等等各种对象，要想让机器把它们一一判断区分开，可就有点难了。一方面，光是要把图片像素转义成“猫”、“狗”这种语义，就十分困难；另一方面，需要分类的对象种类越多，程序员需要设计的规则就越复杂，如果我今天要分类男孩和女孩，就需要指定分类男孩和女孩的规则，而明天要分类小猫和小狗，就又要重新指定分类小猫和小狗的规则，这样显然有点太劳累了，有没有简单点的方法呢？
答案是有的，后来机器学习出现了，它的逻辑是不需要程序员提前设计规则，而是将一定量的训练数据投喂给机器，它能从这些训练数据中总结经验，自己摸索出一套“规则”，然后再用这套“规则”完成新的任务。例如把100张提前人工分类好的小猫和小狗的照片投喂给它，他就会自己总结出一套经验，也就是我们常说的“训练模型”，在此之后，再给它一万张没有分类的照片，它就会按照这套“模型”，出色地完成分类工作。而如果后续还想分类男孩和女孩，也无需再手动修改规则，而是重新投喂给它100张提前分类好的男孩和女孩的照片，经过学习以后，机器又能出色地完成男孩和女孩的分类工作了。

# 2. 机器学习进一步细分
1. 学习范式：根据数据形式和学习目标来划分
	- 监督学习
	- 非监督学习
	- 强化学习
2. 深度学习：使用神经网络作为模型
3. 集成学习
	- Bagging
	- Boosting
	- Stacking
